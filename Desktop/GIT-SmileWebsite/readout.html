About ReadOut

We implemented an optical based fluorescence detection scheme for the LAMP product readout. The intercalating dye SYBR Green 1 was used for the detection of the LAMP product. SYBR Green has a peak excitation at 497nm and a peak emission at 520nm. LAMP readout data can be log in system (PC/Laptop) via USB connector.
